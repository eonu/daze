``daze.plot_confusion_matrix()``
================================

The :func:`plot_confusion_matrix` function is an adjusted version of :func:`sklearn:sklearn.metrics.plot_confusion_matrix`
that adds typical confusion matrix measures (see :doc:`measures`) such as true positives, recall and F1 score to the confusion matrix plot,
making it more informative.

While much of this information is included in the classification report generated by :func:`sklearn.metrics.classification_report`,
it can sometimes be useful to combine these measures alongside confusion matrix, in an easy-to-read format.
