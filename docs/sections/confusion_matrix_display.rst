``daze.ConfusionMatrixDisplay``
===============================

Modification of the :class:`sklearn:sklearn.metrics.ConfusionMatrixDisplay` class which represents a plot of a confusion matrix,
with added :class:`matplotlib:matplotlib.text.Text` objects for evaluation measures and an auto-positioned colorbar.
